# Paradise Nursery Project

## Project Name
Paradise Nursery

## Description
This project is a plant shop web application that allows users to browse products, add them to a cart, and manage orders. The landing page includes a background image and a "Get Started" button.

## Features
- Landing page with company name and background image
- About Us section
- Product listing page
- Shopping cart with add, remove, and quantity update functionality

## Tech Stack
- React.js
- Redux Toolkit
- CSS
